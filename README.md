# NodJs server connecting to MongoDb via API

## MongoDb
### To start the service
```console
brew services start mongodb-community@4.4
```

### To stop the service
```console
brew services stop mongodb-community@4.4
```

### Run NodeJs server
```console
node server
```

## MySql

### Start MySql
```console
mysql.server start
```
Or

```console
brew services start mysql
```

### connect to MySql using binary
```console
mysql -u root -p
Enter password:******
```

### Disconnect MySql
```console
mysql> exit
Bye
```
### Stop MySql
```console
mysql.server stop
```
Or

```console
brew services stop mysql
```
